# Ethiopia FAO Policy Mapping Dashboard

An interactive dashboard for exploring Ethiopia's policy alignment with the FAO Global Roadmap.

## Features

- **10 FAO Domains**: Visualized as an interactive donut chart
- **129 Roadmap Actions**: Detailed breakdown by domain
- **532 Policy Actions**: Linked to policy documents with source references
- **548 Metrics**: Target, baseline, and target year data
- **Smart Search**: Find actions by keyword across all domains
- **Document Links**: Direct access to 36 policy documents via FAOLEX and other sources

## How to Use

1. **Explore Domains**: Hover over the donut chart to see domain statistics
2. **Drill Down**: Click on any domain to view its roadmap actions
3. **View Details**: Select a roadmap action to see associated metrics and policy actions
4. **Search**: Use the search bar to find specific actions by keyword

## Data Source

FAO Ethiopia Policy Document Analysis - Last updated: November 2024

## Technical Details

- Built with React 18 and Tailwind CSS
- Interactive SVG visualizations
- Responsive design with dark theme
- No backend required - runs entirely in the browser
